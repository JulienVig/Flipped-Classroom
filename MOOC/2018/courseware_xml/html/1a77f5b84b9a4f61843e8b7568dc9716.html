Contenu de la leçon

Dans cette leçon, nous verrons qu'il est possible de décomposer deux matrices en "plus petites matrices", dans le but de simplifier l'addition ou la multiplication desdites matrices.

objectifs

Une fois la matière de cette leçon acquise et les quiz et exercices correspondants maîtrisés, l'étudiant devrait être capable

  1. de déterminer s'il est possible d'additionner/multiplier par blocs deux matrices dont les décompositions en blocs sont données  ;
  2. d'effectuer l'addition par blocs, lorsque celle-ci est possible ;
  3. d'effectuer la multiplication par blocs, lorsque celle-ci est possible.
