Contenu de la leçon

Dans cette leçon, nous introduirons les concepts de familles orthogonales/orthonormales et verrons que lorsque qu'une telle famille forme une base de l'espace vectoriel étudié, alors il est facile de déterminer les coordonnées d'un vecteur donné par rapport à celle-ci !

objectifs

Une fois la matière de cette leçon acquise et les quiz et exercices correspondants maîtrisés, l'étudiant devrait être capable

  1. de déterminer si une collection (finie) de vecteurs est orthogonale (ou orthonormale) ;
  2. de calculer les coordonnées d'un vecteur donné dans une base orthogonale (ou orthonormale) sans effectuer de combinaisons linéaires.
