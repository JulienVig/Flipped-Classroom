Contenu de la leçon

Dans cette leçon, nous introduirons une opération sur l'ensemble des matrices, appelée la multiplication matricielle. Aussi, nous étudierons le comportement qu'entretient cette multiplication avec les opérations précédemment introduites, comme la transposée, ou l'addition matricielle, par exemple.

objectifs

Une fois la matière de cette leçon acquise et les quiz et exercices correspondants maîtrisés, l'étudiant devrait être capable

  1. de définir la multiplication matricielle ;
  2. de déterminer s'il est possible de multiplier deux matrices entre elles et d'effectuer ladite multiplication dans le cas échéant ;
  3. de maîtriser les différentes propriétés correspondantes (voir l'onglet Concept(s)-clé(s) pour un récapitulatif).
