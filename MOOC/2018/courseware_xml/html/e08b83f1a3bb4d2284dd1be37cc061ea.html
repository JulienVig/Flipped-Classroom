Contenu de la leçon

Dans cette leçon, nous montrerons comment appliquer le critère de diagonalisabilité introduit dans la vidéo précédente à l'aide de plusieurs exemples, traités en détail. Dans ces divers exemples, nous verrons comment déterminer si la transformation linéaire (ou matrice) est diagonalisable à l'aide dudit critère, puis lorsque c'est le cas, nous montrerons comment trouver le changement de base adéquat.

objectifs

Une fois la matière de cette leçon acquise et les quiz et exercices correspondants maîtrisés, l'étudiant devrait être capable

  1. de déterminer si une transformation linéaire est diagonalisable ou non, en utilisant le critère vu dans la leçon ;
  2. de déterminer la matrice de changement de base permettant de diagonaliser ladite transformation, lorsque celle-ci est diagonalisable.
