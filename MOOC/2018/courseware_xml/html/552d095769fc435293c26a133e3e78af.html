Contenu de la leçon

Dans cette leçon, nous définirons le concept de coordonnées d'un vecteur par rapport à une base ordonnée. Nous remarquerons aussi que ces coordonnées se comportent de manière idéale par rapport aux combinaisons linéaires.

objectifs

Une fois la matière de cette leçon acquise et les quiz et exercices correspondants maîtrisés, l'étudiant devrait être capable

  1. de calculer les coordonnées ainsi que le vecteur des coordonnées d'un vecteur dans une base ordonnée ;
  2. de déduire les coordonnées d'une combinaison linéaire de vecteurs à partir des coordonnées de ces derniers.
