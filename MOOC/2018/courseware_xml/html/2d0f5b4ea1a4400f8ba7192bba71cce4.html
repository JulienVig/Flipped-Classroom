Concept(s)-clé(S)

LEMME 1 :

Soient \(V\) un \(\mathbb{R}\)-espace vectoriel admettant une base \(\mathscr{B}=(v_1,\ldots,v_n)\) et \(W\) un \(\mathbb{R}\)-espace vectoriel arbitraire. Soient encore \(w_1,\ldots,w_n\in W.\) Alors l'application \(T:V\to W\) définie par \[T(a_1v_1+\cdots+a_nv_n)=a_1w_1+\cdots+a_nw_n, ~(\forall ~a_1,\ldots,a_n\in \mathbb{R})\] est linéaire.

 

LEMME 2 :

Soit \(V\) un \(\mathbb{R}\)-espace vectoriel admettant une base \(\mathscr{B}=(v_1,\ldots,v_n).\)  Alors l'application \(T:V\to \mathbb{R}^n\) définie par \[T(a_1v_1+\cdots+a_nv_n)=(a_1,\ldots,a_n), ~(\forall ~a_1,\ldots,a_n\in \mathbb{R})\] est linéaire.

 

DÉFINITION 3 :

Soient \(V\) un \(\mathbb{R}\)-espace vectoriel de dimension \(n\) et \(W\) un \(\mathbb{R}\)-espace vectoriel de dimension \(m\). Soient \(\mathscr{B}_V\) une base de \(V\) et \(\mathscr{B}_W\) une base de \(W.\) Finalement, considérons \(A\in M_{m\times n}(\mathbb{R})\) une matrice de taille \(m\times n\) à coefficients réels. On définit une application \(T_A : V\to W\) par \[[T_A(v)]_{\mathscr{B}_W}= A[v]_{\mathscr{B}_V},\] ceci pour tout \(v\in V.\)
