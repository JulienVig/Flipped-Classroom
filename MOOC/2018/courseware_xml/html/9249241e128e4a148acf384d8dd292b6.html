Concept(s)-clé(S)

PROPOSITION :

Soient \(A\in M_{m\times n}(\mathbb{R})\) et \(\hat{A}\) une matrice échelonnée ligne-équivalente à \(A.\) Si les pivots de \(\hat{A}\) se situent dans les colonnes aux indices \(i_1,\ldots, i_t\) de \(\hat{A},\) alors les colonnes \(C_{i_1},\ldots,C_{i_t}\) de \(A\) forment une base de l'espace-colonnes de \(A.\)
