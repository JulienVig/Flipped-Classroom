Contenu de la leçon

Dans cette leçon, nous commencerons par comparer deux matrices carrées entre elles, pour voir que bien que possédant le même polynôme caractéristique, l'une d'elles est diagonalisable, mais pas l'autre ! En nous inspirant de ce qu'il est possible de déduire des multiplicités géométriques des valeurs propres (communes) de ces deux matrices, nous démontrerons un théorème fondamental de ce cours : un critère de diagonalisabilité se basant sur les multiplicités géométriques des diverses valeurs propres.

objectifs

Une fois la matière de cette leçon acquise et les quiz et exercices correspondants maîtrisés, l'étudiant devrait être capable

  1. de déterminer si une matrice donnée est diagonalisable ou non, en utilisant le critère vu dans la leçon ;
  2. de déterminer si une transformation linéaire donnée est diagonalisable ou non, en utilisant le critère vu dans la leçon.
