Concept(s)-clé(S)

DÉFINITION 1 :

Soient \(V,W\) deux \(\mathbb{R}\)-espaces vectoriels et \(T:V\to W\) une application linéaire. L'image de \(T\) est le sous-ensemble de \(W\) défini par \[\mbox{im}(T)=\{T(v):v\in V\}.\]

PROPOSITION 2 :

Soient \(V,W\) deux \(\mathbb{R}\)-espaces vectoriels et \(T:V\to W\) une application linéaire. Alors \(\mbox{im}(T)\) est un sous-espace vectoriel de \(W.\)

DÉFINITION 3 :

Soient \(V,W\) deux \(\mathbb{R}\)-espaces vectoriels et \(T:V\to W\) une application linéaire. Si \(\mbox{im}(T)\) est de dimension finie, alors on appelle l'entier \(\mbox{dim} (\mbox{im} (T))\) le rang de \(T.\)
