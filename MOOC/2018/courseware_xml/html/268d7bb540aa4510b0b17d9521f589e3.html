Concept(s)-clé(S)

DÉFINITION :

Une transformation linéaire \(\phi:V\to V\) d'un \(\mathbb{R}\)-espace vectoriel de dimension finie \(V\) est diagonalisable s'il existe une base de \(V\) formée de vecteurs propres pour \(\phi.\) Aussi, une matrice \(A\in M_{n\times n}(\mathbb{R})\) est dite diagonalisable s'il existe \(P\in M_{n\times n}(\mathbb{R})\) inversible telle que \(P^{-1} A P\) soit diagonale.
