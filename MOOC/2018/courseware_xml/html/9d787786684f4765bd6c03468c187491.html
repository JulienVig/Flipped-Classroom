Concept(s)-clé(S)

DÉFINITION :

Soit \(A\in M_{m\times n}(\mathbb{R})\) une matrice dont les colonnes sont linéairement indépendantes (vues comme vecteurs de \(\mathbb{R}^m\). Alors il existe une factorisation du type \(A=QR,\) où \(Q\) est une matrice \(m\times n\) dont les colonnes forment une base orthonormée de l'espace colonnes de \(A\) et \(R\) est une matrice triangulaire supérieure, inversible, dont les coefficients diagonaux sont strictement positifs.

ALGORITHME :

Soit \(A\in M_{m\times n}(\mathbb{R})\) une matrice dont les colonnes sont linéairement indépendantes (vues comme vecteurs de \(\mathbb{R}^m\). Afin de déterminer une factorisation \(QR\) de \(A,\) on procède comme suit :

  1. Poser \(\mathscr{C}=(c_1,\ldots,c_n)\), qui ici forme une base de l'espace colonnes \(W\) de \(A\).
  2. A l'aide du procédé de Gram-Schmidt, calculer une base orthonormée \(\mathscr{B}=(w_1,\ldots,w_n)\) de \(W\).
  3. Définir \(Q\in M_{m\times n}(\mathbb{R})\) comme étant la matrice dont la \(i\)-ème colonne est \(w_i\).
  4. Pour tout \(1\leq k\leq n,\) écrire \(c_k=r_{1k}w_k + r_{2k}w_2 +\cdots + r_{kk}w_k + 0w_{k+1} +\cdots + 0 w_n.\) (On supposera que \(r_{ii}\geq 0,\) quitte à remplacer \(w_i\) par \(-w_i\).) Poser alors \[r_k=\begin{pmatrix} r_{1k} \cr r_{2k} \cr \vdots \cr r_{kk} \cr 0 \cr \vdots \cr 0\end{pmatrix}.\]
  5. Définir \(R\in M_{n\times n}(\mathbb{R})\) comme étant la matrice dont la \(i\)-ème colonne est \(r_i.\)
