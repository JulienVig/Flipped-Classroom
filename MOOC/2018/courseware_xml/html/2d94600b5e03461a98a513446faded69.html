Forum de discusion : Présentation

Le Forum est subdivisé en 5 catégories, dont les descriptions sont les suivantes.

  1. La catégorie Général/Social vous donne la possibilité de vous présenter, de converser avec d'autres étudiants, etc. Celle-ci ne sera pas lue par le personnel enseignant et donc ne sera pas modérée.
  2. La catégorie Questions sur les leçons vous permet de poster des questions/remarques concernant les leçons, c'est-à-dire principalement les vidéos de cours et les sections associées : Contenu/Objectif(s), Concept(s)-clef(s).
  3. Dans la catégorie Questions sur les quiz, vous pouvez poster des questions/remarques concernant les quiz des différents chapitres.
  4. La catégorie Questions sur les exercices online accueillera les questions/remarques sur les exercices online (qu'il est nécessaire de compléter si vous désirez l'obtention d'un certificat d'accomplissement).
  5. Finalement, dans la catégorie Questions sur les séries en format PDF, vous avez la possibilité de poster des questions/remarques concernant les séries en format PDF ainsi que leurs corrigés (PDF et vidéos).

Forum de discusion : organisation

Afin d'assurer un bon fonctionnement du forum de discussion ainsi qu'une bonne ambiance sur celui-ci, nous vous prions de respecter les règles suivantes :

  1. Lorsque vous postez une question/remarque concernant une vidéo, un quiz, etc., veuillez s'il vous plaît indiquer dans le titre de votre post le chapitre, la section, ainsi que le numéro de vidéo/quiz/exercice auxquels vous vous référez.
  2. Lorsque vous avez l'intention de nous faire remarquer une erreur mathématique dans une vidéo, un quiz, un exercice, etc., veuillez écrire [ERREUR] dans le titre de votre post. Un tel post sera traité en priorité et s'il s'avère qu'une erreur est en effet présente à l'endroit mentionné, alors elle sera corrigée aussi vite que possible.
  3. Il vous est possible de poser vos questions en français ou en anglais.
  4. Bien entendu, veuillez vous montrer respectueux les uns envers les autres et écrivez de manière convenable, de façon à rendre les discussions autant agréables que possible.
