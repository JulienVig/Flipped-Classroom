Contenu de la leçon

Dans cette leçon, nous verrons qu'il est possible de réduire de moitié le travail nécessaire afin de montrer qu'une collection de vecteurs dans un espace vectoriel de dimension finie (important!) est une base, lorsque cette dimension est connue.

objectifs

Une fois la matière de cette leçon acquise et les quiz et exercices correspondants maîtrisés, l'étudiant devrait être capable d'utiliser le résultat vu en cours afin de montrer qu'une collection de vecteurs dans un espace vectoriel de dimension finie connue est une base.
