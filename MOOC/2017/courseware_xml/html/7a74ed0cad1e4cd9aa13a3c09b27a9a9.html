Application : exercices online

La section (ou les sections, comme dans le chapitre 4, par exemple) dédiée(s) aux exercices online. Ces derniers, souvent classés par thèmes (Interprétation géométrique, Résolution de systèmes linéaires, Matrices et systèmes linéaires dans la section 1.9, par exemple) forment l'entièreté de la partie évaluée du MOOC. Comme déjà mentionné, le résultat n'a aucune influence sur l'examen d'Algèbre Linéaire de janvier à l'EPFL. Voici donc les points importants.

  1. Les exercices online doivent être validés dans les délais chaque semaine.
  2. Le nombre total de points qu'il est possible d'obtenir lors de la complétion d'un exercice est inscrit en haut de ce dernier.
  3. Une fois un exercice validé, le nombre de points obtenus s'affichera à côté du nombre total de points. Vous verrez également si ledit exercice a été effectué correctement ou non et il vous sera possible d'accéder à la réponse d'un simple clic !
  4. Comme les réponses vous sont accessibles immédiatement après validation de vos choix, un seul essai est autorisé par exercice. Ainsi, une fois un exercice validé, il n'est plus possible de revenir en arrière dans le but d'améliorer votre score.

Remarque importante concernant les questions à choix multiples :

Les questions à choix multiples sont principalement de deux types : certaines peuvent admettre plus d'une solution, tandis que d'autres n'en n'admette qu'une seule. Cela peut se voir aisément de la façon suivante.

  1. Lorsque les cases à cocher sont rondes, il n'y a qu'une seule solution à l'exercice.
  2. Quand les cases à cocher sont carrées, alors il peut y avoir plusieurs réponses possibles, auquel cas il est impératif de toutes les cocher afin d'obtenir tous les points.
