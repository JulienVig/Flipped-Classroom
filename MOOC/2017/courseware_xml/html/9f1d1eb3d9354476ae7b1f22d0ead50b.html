Contenu de la leçon

Dans cette leçon, nous commencerons par définir ce qu'est le polynôme caractéristique d'une matrice carrée. Ensuite, nous démontrerons que celui-ci est un invariant de similitude (c'est-à-dire, qu'il est exactement le même pour deux matrices semblables). Ceci nous permettra de définir ce qu'est le polynôme caractéristique d'une transformation linéaire. Nous conclurons par un exemple illustratif.

objectifs

Une fois la matière de cette leçon acquise et les quiz et exercices correspondants maîtrisés, l'étudiant devrait être capable

  1. de définir précisément ce qu'est le polynôme caractéristique d'une matrice carrée ;
  2. de définir précisément ce qu'est le polynôme caractéristique d'une transformation linéaire \(T:V\to V\) et d'expliquer pourquoi la définition ne dépend pas de la base de \(V\) choisie ;
  3. de calculer le polynôme caractéristique d'une transformation linéaire (simple) donnée.
