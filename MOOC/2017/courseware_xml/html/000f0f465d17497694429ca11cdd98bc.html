Contenu de la leçon

Dans cette leçon, nous verrons comment il est possible d'appliquer les notions de coordonnées et de rang-ligne introduites dans le chapitre afin de construire efficacement une base d'un espace vectoriel donné à partir d'un système de générateurs.

objectifs

Une fois la matière de cette leçon acquise et les quiz et exercices correspondants maîtrisés, l'étudiant devrait être capable

  1. de trouver une base d'un sous-espace vectoriel engendré par une collection connue de vecteurs en appliquant la méthode introduite en cours ;
  2. de compléter la base précédemment déterminée en une base de l'espace vectoriel.
