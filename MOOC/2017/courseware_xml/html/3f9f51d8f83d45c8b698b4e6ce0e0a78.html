Contenu de la leçon

Dans cette leçon, nous verrons (à l'aide d'un exemple) qu'il peut être avantageux de considérer d'autres bases que les bases canoniques lorsque l'on veut étudier matriciellement une application linéaire.

 

objectifs

Une fois la matière de cette leçon acquise et les quiz et exercices correspondants maîtrisés, l'étudiant devrait être capable

  1. d'expliquer pourquoi il peut être avantageux de considérer la matrice d'une application linéaire dans d'autres bases que les bases canoniques ;
  2. de donner un exemple concret (pas forcément celui vu en cours) d'une situation dans laquelle un changement de base pourrait être bénéfique.
