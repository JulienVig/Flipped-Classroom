Application : SÉRIES PDF

Dans cet onglet, nous présenterons brièvement la section (ou les sections, comme dans le chapitre 4, par exemple) dédiée(s) aux séries d'exercices PDF. Celles-ci ne sont pas évaluées, mais constituent une opportunité de vous entraîner avant de compléter les séries online, qui elles, forment l'intégralité du contenu évalué du cours (voir onglet Application : exercices online pour plus de détails). Les points importants concernant les séries PDF sont les suivants :

  1. Un corrigé très détaillé de chacune de ces séries est disponible (au même endroit que ladite série).
  2. Certains problèmes bénéficient d'une correction détaillée en vidéo, dans laquelle un des enseignants procédera à leur résolution.
  3. Ces séries ne sont pas évaluées.
